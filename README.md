# 二开版海外抢单系统/抢单刷单系统/分组联单控/订单自动匹配

此套是别人二开的海外抢单系统，系统支持分组联单控制、入职生成、订单编辑续期解冻等功能<br>系统前端多语言自带多个语言包可以自行添加，后台支持设置默认语言和关闭前端语言选择开关<br>
